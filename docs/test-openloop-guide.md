## 故障排查

### 电机不转，PWM 无电压

**可能原因：**
1. TIM1 未正确初始化
2. PWM 未启动
3. GPIO 时钟未使能

**检查方法：**
```c
// 在 Mecanum_Init() 后添加测试代码
DC4_Motor_SetSignedSpeed(0, 50);  // 强制前左电机50%速度
HAL_Delay(1000);
```

### 电机不转，PWM 有电压

**可能原因：**
1. TB6612 供电问题
2. 方向引脚接线错误
3. 电机驱动板故障

**检查方法：**
1. 测量 TB6612 VM 引脚电压（应为电池电压）
2. 测量 STBY 引脚（应为高电平）
3. 检查 IN1/IN2 逻辑电平

### 蓝牙无响应

**可能原因：**
1. 蓝牙模块未配对
2. 波特率不匹配（应为9600）
3. USART3 配置错误

## 预期结果

根据代码分析：

1. **前进时：**
   - 所有电机正转
   - IN1=HIGH, IN2=LOW
   - PWM 占空比 ≈ 50% (约1.65V)

2. **后退时：**
   - 所有电机反转
   - IN1=LOW, IN2=HIGH
   - PWM 占空比 ≈ 50%

3. **停止时：**
   - IN1=HIGH, IN2=HIGH（制动模式）
   - PWM = 0V
