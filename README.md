# omnibot-f4 — STM32F4 全向机器人

基于 STM32F407 的全向机器人平台，集成**四轮麦克纳姆轮小车**、**步进电机升降**与**双电磁阀夹爪**，支持蓝牙遥控与实时调参。

## 硬件概览

| 模块 | 型号 |
|------|------|
| 主控 | STM32F407VGT6 (168MHz) |
| 电机驱动 | TB6612 四通道 |
| 步进驱动 | X42S 双轴 (485 总线) |
| 电磁阀 | 双路 (PE10 / PE6) |
| 蓝牙 | HC-05/06 (USART3, 9600bps) |
| 编码器 | 四路增量式编码器 |

### 引脚分配

**TB6612 电机驱动：**

| 电机 | PWM | IN1 | IN2 |
|------|-----|-----|-----|
| 前右（实际 idx0） | PA2 (TIM5_CH3) | PB0 | PB1 |
| 前左（实际 idx1） | PA3 (TIM5_CH4) | PF13 | PF14 |
| 后左（实际 idx2） | PE11 (TIM1_CH2) | PE12 | PE9 |
| 后右（实际 idx3） | PE13 (TIM1_CH3) | PA5 | PA4 |

**编码器：**

| 电机 | A相 | B相 | 定时器 |
|------|-----|-----|--------|
| 前左 | PA0 | PA1 | TIM2 |
| 前右 | PA6 | PA7 | TIM3 |
| 后左 | PB6 | PB7 | TIM4 |
| 后右 | PC6 | PC7 | TIM8 |

**X42S 步进驱动器：** USART1（485 总线，双电机并接）

**电磁阀：**

| 阀 | 引脚 | 功能 |
|----|------|------|
| 阀1 | PE10 | 夹爪开闭 |
| 阀2 | PE6 | 夹爪前后伸缩 |

## 软件架构

| 文件 | 说明 |
|------|------|
| `motor_driver_dc4ch.c` | 四路直流电机 PWM + 编码器驱动 |
| `motor_closedloop.c` | PID 闭环速度控制 (10ms 周期) |
| `motor_driver_X42S.c` | X42S 步进电机串口控制 |
| `solenoid_valve.c` | 双路电磁阀控制 |
| `app_control.c` | 麦轮运动学解算 + 蓝牙协议解析 |
| `bluetooth.c` | 蓝牙收发 (USART3, DMA) |

## 功能特性

- 四轮麦克纳姆轮全向运动
- PID 闭环速度控制 + 蓝牙实时调参
- X42S 步进电机夹爪升降
- 双电磁阀夹爪开闭 / 前后伸缩
- 蓝牙绘图调试 (VOFA+ 协议)
- 电机状态实时查询

## 蓝牙协议

### 麦轮运动 `[joystick,lx,ly,rx,ry]`

| 参数 | 说明 | 范围 |
|------|------|------|
| lx | 左右平移 | -100 ~ 100 |
| ly | 前后移动 | -100 ~ 100 |
| rx | 旋转 | -100 ~ 100 |

示例：`[joystick,0,50,0,0]` 前进，`[joystick,50,0,0,0]` 右平移

### 步进升降 `[gripper,x_speed,y_speed]`

| 参数 | 说明 | 范围 |
|------|------|------|
| x_speed | 升降速度 | -300 ~ 300 RPM |
| y_speed | （预留） | -300 ~ 300 RPM |

示例：`[gripper,200,0]` 上升

### PID 调参 `[pid,电机索引,Kp*100,Ki*100,Kd*100]`

电机索引：0~3 对应四个电机，4 表示全部。

示例：`[pid,4,50,10,0]` → Kp=0.5, Ki=0.1, Kd=0

### 绘图调试

| 命令 | 说明 |
|------|------|
| `[auto]` | 自动连续发送速度数据 (100ms) |
| `[plot]` / `[p]` | 发送一次数据 |
| `[plot-clear]` / `[p-c]` | 清空绘图区 |
| `[stop]` | 停止自动发送 |

### 电磁阀控制

#### 阀1（PE10）— 夹爪开闭

| 命令 | 说明 |
|------|------|
| `[valve1,on]` | 打开（夹爪闭合） |
| `[valve1,off]` | 关闭（夹爪张开） |
| `[valve1,toggle]` | 切换开关状态 |
| `[valve1,pulse,500]` | 打开 500ms 后自动关闭（范围 1~60000ms） |
| `[valve1,query]` | 查询当前开关状态 |

#### 阀2（PE6）— 夹爪前后伸缩

| 命令 | 说明 |
|------|------|
| `[valve2,on]` | 打开（夹爪伸出） |
| `[valve2,off]` | 关闭（夹爪缩回） |
| `[valve2,toggle]` | 切换开关状态 |
| `[valve2,pulse,500]` | 打开 500ms 后自动关闭（范围 1~60000ms） |
| `[valve2,query]` | 查询当前开关状态 |

## 快速开始

1. 用 Keil MDK 打开 `MDK-ARM/方舟.uvprojx`，编译烧录
2. 手机蓝牙串口 APP 连接 HC-05/06
3. 发送命令控制小车 / 夹爪
