# 方舟 - STM32 麦克纳姆轮小车

基于 STM32F4 的四轮麦克纳姆轮小车，支持蓝牙遥控和 PID 闭环速度控制。

## 硬件配置

| 模块 | 型号/引脚 |
|------|----------|
| 主控 | STM32F4xx (168MHz) |
| 电机驱动 | TB6612 四通道 |
| 蓝牙 | HC-05/06 (USART3, 9600波特) |
| 编码器 | 四路增量式编码器 |

### 引脚分配

**TB6612 电机驱动：**
| 电机 | PWM引脚 | IN1 | IN2 |
|------|---------|-----|-----|
| 前左 | PE9 | PB0 | PB1 |
| 前右 | PE11 | PF13 | PF14 |
| 后左 | PE13 | PE7 | PE12 |
| 后右 | PE14 | PA5 | PA4 |

**编码器：**
| 电机 | A相 | B相 | 定时器 |
|------|-----|-----|--------|
| 前左 | PA0 | PA1 | TIM2 |
| 前右 | PA6 | PA7 | TIM3 |
| 后左 | PB6 | PB7 | TIM4 |
| 后右 | PC6 | PC7 | TIM8 |

## 功能特性

- 四轮麦克纳姆轮全向运动控制
- PID 闭环速度控制（10ms 周期）
- 蓝牙遥控（摇杆模式）
- 蓝牙实时调参（PID 参数）
- 蓝牙绘图功能（调试用）
- 电机状态查询

## 蓝牙协议

所有数据包格式：`[类型,参数1,参数2,...]`

### 运动控制

```
[joystick,lx,ly,rx,ry]
```

| 参数 | 说明 | 范围 |
|------|------|------|
| lx | 左右平移 | -100 ~ 100 |
| ly | 前后移动 | -100 ~ 100 |
| rx | 旋转 | -100 ~ 100 |
| ry | 未使用 | - |

示例：
- 前进：`[joystick,0,50,0,0]`
- 后退：`[joystick,0,-50,0,0]`
- 右平移：`[joystick,50,0,0,0]`
- 原地右转：`[joystick,0,0,50,0]`
- 停止：`[joystick,0,0,0,0]`

### PID 调参

```
[pid,电机索引,Kp*100,Ki*100,Kd*100]
```

| 电机索引 | 说明 |
|---------|------|
| 0 | 前左 |
| 1 | 前右 |
| 2 | 后左 |
| 3 | 后右 |
| 4 | 全部 |

示例：
- 设置全部电机：`[pid,4,50,10,0]`（Kp=0.5, Ki=0.1, Kd=0）
- 设置前左电机：`[pid,0,80,20,0]`

### 状态查询

```
[query]
```

返回：`[s,速度0,速度1,速度2,速度3]`

### 绘图功能

| 命令 | 说明 |
|------|------|
| `[plot]` 或 `[p]` | 发送一次电机速度数据 |
| `[plot-clear]` 或 `[p-c]` | 清空绘图区 |
| `[auto]` | 开始自动连续发送（100ms间隔） |
| `[stop]` | 停止自动发送 |

绘图数据格式：`[p,速度0,速度1,速度2,速度3]`

## 使用方法

1. 烧录程序到 STM32
2. 手机蓝牙串口 APP 连接 HC-05/06
3. 发送命令控制小车

### 调 PID 步骤

1. 开始绘图：发送 `[auto]`
2. 让车走：发送 `[joystick,0,50,0,0]`
3. 观察波形，调 PID：发送 `[pid,4,60,20,0]`
4. 重复直到满意
5. 停止：发送 `[stop]`
